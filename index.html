PHP: CONNECTING TO DATABASE WITH PDO PREPARED STATEMENTS

Code Description

This class simplifies the the connection to a database using query(), bind(), execute(), fetchSingle(), fetchMultiple() and fetchCount()

How this code works?

When you create a new object new dbase;, it tries to connect to the database. The MySQL query is then passed into the function query(). The function bind() is then used to bind any variables specified into the query. Finally, we use execute() to run the statement.

query('SQL STATEMENTS');
bind(':variable','value',PDO::PARAM_STR);
execute();

PHP: REMOVE EVERYTHING IN A STRING THAT COMES BEFORE/ AFTER A CHARACTER, INCLUDING THE CHARACTER

Code Description

This can be achieved using string manipulation functions in PHP. First we find the position of the desired character in the string using strpos(), substr(), ltrim, and trim()

Before

After

How this code works?

The above code was used to fix a problem with the global variable $_GET[] when calling pages on the index.php file for our back-end applications. The code was used to check the string in the url after index.php? by removing the question sign (?) before it and any other sign after it primarily an equal sign or an ampersand after it. The global variable $_SERVER["REQUEST_URI"] was used to grab the url. The process required two stages.

Part 1: Remove question mark and everything before it 

            
//get ur
//global variable:
$_SERVER["REQUEST_URI"] = "/biz/cuc/index.php?studentprofile=C3836B2C";
        
$question_mark_check = ltrim(stristr($_SERVER["REQUEST_URI"], '?'), '?')
//output: 
studentprofile=C3836B2C

Part 2: Remove equal sign and everything after it from the previous code executed 

            
//set input variable

$equal_sign_check = substr($question_mark_check, strpos($question-mark_check, '=') + 1)

Issues implementing this code

There was one issue encountered when using this code. The code returns an empty value if the url does not have an equal sign or the ampersand in the string. To resolve this an if statement was used.

            
                
function getSlug() {        
    //get url from server and asign it to a variable
    $url = $_SERVER["REQUEST_URI"];
    
    //remove the last string in in url after the & sign
    $removed_last_string = substr($url, 0, strpos($url, "&"));
   
    //check to see if there was an & sign was present
    if(!empty($removed_last_string)){

    //if yes, remove the ? from the beginning of the string
    $almost_cleaned_string = ltrim(stristr($removed_last_string, '?') , '?');
       
        //check to see if the string contains an equal sign
        if(strpos($almost_cleaned_string, '=')){

        //if yes, remove it 
            $cleaned_string =  substr($almost_cleaned_string, 0, strpos($almost_cleaned_string, "="));
            return $cleaned_string;
        }else{
       
        //if not, then return the previously sanitized string 
        $cleaned_string = $almost_cleaned_string;
        return $cleaned_string;
        }
   
    }
    //check to see if there was not & sign was present
    elseif(empty($removed_last_string)){

        //check to see if the string contains an equal sign 
        $almost_cleaned_string = substr($url, 0, strpos($url, "="));
       
        //then check to see if it produced an empty result
        if(!empty($almost_cleaned_string)){
            
            //if no, remove the question mark at the beginning and return the sanitized string
            $cleaned_string = ltrim(stristr($almost_cleaned_string, '?') , '?');
            return $cleaned_string;

        }else{

            //if yes, remove the question mark at the beginning of the original url string
            $cleaned_string = ltrim(stristr($url, '?') , '?');
            return $cleaned_string;
        }
    }
   
}

PHP: Understanding Arrays vs Strings

Index Arrays

The first type of array is the index array. Its reference is a numerical value 

        
$array1a = array("Math","Teacher","IT");

//OUTPUT: 
Array
(
    [0] => Math
    [1] => Teacher
    [2] => IT
)

or

        
$array1b = ["Math","Teacher","IT"];
//OUTPUT: 
Array
(
    [0] => Math
    [1] => Teacher
    [2] => IT
)

Associative Arrays

The second type of array is the associative array. Its value is referenced by key. This key can be an integer or a string

        
$array1c = array("Math" => "3","Teacher" => "37" ,"IT" => "4");

//OUTPUT:
Array
(
    [Math] => 3
    [Teacher] => 37
    [IT] => 4
)

or

        
$array1d = ["Math" => "3","Teacher" => "37" ,"IT" => "4"];
//OUTPUT:
Array
(
    [Math] => 3
    [Teacher] => 37
    [IT] => 4
)

Strings

The a string is a series of characters enclosed in single or double quote.

        
$str1 = "Math,Teacher,IT";

or

         
$str2 = '["Math","Teacher","IT"]';

or

            
$str3 = '{"Math":3,"Teachers":37,"IT":4}';

Converting A String To An Array

To convert a string to an array it is important to note the format of the string. Knowing the format of the string we can determine if it will convert to an index array or an associative array.

Converting To An Index Array

            
$str1 = "Math,Teacher,IT";

explode(",",$str1);

//OUTPUT:
Array
(
    [0] => Math
    [1] => Teacher
    [2] => IT
)
        
    
        
$str2 = '["Math","Teacher","IT"]';
    
json_decode($str2);
    
//OUTPUT:
Array
(
    [0] => Math
    [1] => Teacher
    [2] => IT
)

The above strings can only be converted to an index array. This can be easily done by using the explode() function

Converting A JSON String To An Associative Array

            
$str4 = '{"Math":3,"Teachers":37,"IT":4}';
        
json_decode($str3);
        
//OUTPUT:
stdClass Object
